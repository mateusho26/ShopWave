# 🛍️ ShopWave

Projeto desenvolvido para a disciplina de Desenvolvimento Front-end utilizando **React + JavaScript**.

## 📖 Sobre o projeto

O ShopWave é uma loja virtual que consome dados da Fake Store API, permitindo visualizar produtos, pesquisar por nome, filtrar por categoria e adicionar itens ao carrinho.

## 🚀 Funcionalidades

- Listagem de produtos
- Busca por nome
- Filtro por categoria
- Carrinho de compras
- Loading durante o carregamento
- Tratamento de erros da API
- Interface responsiva

## 🛠️ Tecnologias utilizadas

- React
- JavaScript
- Vite
- CSS3
- Fake Store API

## 📦 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/shopwave.git
```

Entre na pasta:

```bash
cd shopwave
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
